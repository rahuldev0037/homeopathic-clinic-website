# 🏥 Homeopathic Clinic Website

![PHP](https://img.shields.io/badge/PHP-Core_PHP-blue)
![MySQL](https://img.shields.io/badge/Database-MySQL-orange)
![Bootstrap](https://img.shields.io/badge/Frontend-Bootstrap-purple)
![Status](https://img.shields.io/badge/Status-Completed-success)

A responsive healthcare website developed using **Core PHP** and **MySQL**. The application allows patients to explore treatments, contact the clinic, and book appointments through a clean and user-friendly interface.

# ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/rahuldev0037/homeopathic-clinic-website.git
```

### Move Project

Copy project into

```
xampp/htdocs/
```

### Import Database

- Open phpMyAdmin
- Create database
- Import SQL file

### Run Project

```
http://localhost/homeopathic-clinic-website/
```
